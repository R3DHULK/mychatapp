# My Chat App

## A Chat App Written In React And Node Js.

# Git Installation

```
git clone https://github.com/R3DHULK/mychatapp
cd mychatapp
```

# To Start Fronted
```
cd app
npm i
npm run dev
```

# To Run Backend
```
cd server
npm i
node index.js
```
